A mobile device software component that must be installed on an [Android](/automation-guide/application-testing/mobile-testing/testing-mobile-applications/android-automation/setting-up-the-test-environment/setting-up-android-automation/about-testarchitect-agent-in-android/installing-testarchitect-agent) or [iOS](/automation-guide/application-testing/mobile-testing/testing-mobile-applications/ios-automation/setting-up-the-ios-test-environment/setting-up-ios-automation/re-signing-and-installing-testarchitect-agent) to allow TestArchitect to conduct automated testing on the device.

Also referred to as: **TA Agent**.

{{<note>}}

-   Do not confuse with [TestArchitect Automation Agent](/user-guide/support/glossary-of-terms/testarchitect-automation-agent), which is used for web testing, nor with [TestArchitect agent license](/user-guide/support/glossary-of-terms/testarchitect-agent-license), which grants rights to the TestArchitect Controller service.
