# TestArchitect is unable to find the specified item. Please check the *<itemName\>* item in the *<controlName\>* control, which resides in the *<windowName\>* window, exists. {#reference_0x80010402L .reference}

Error code: 0x80010402L

## Problem { .section}

This happens because the specified item does not exist.

## Solution { .section}

Ensure that the given item exists, or try a different item, and then re-run the test.
