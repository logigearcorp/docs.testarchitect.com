# TestArchitect is unable to find the specified menu item. Please check that the *<itemValue\>* menu item, which resides in the *<windowName\>* window, exists. {#reference_0x80010A02L_1 .reference}

Error code: 0x80010A02L

## Problem { .section}

This happens because the given menu item does not exist.

## Solution { .section}

Ensure that the menu item exists, and then re-run the test.
