# TestArchitect is unable to find the specified column header. Please check the that column header at position *<headerPosition\>* in the *<controlName\>* control, which resides in the *<windowName\>* window, exists. {#reference_0x80016004L .reference}

Error code: 0x80016004L

## Problem { .section}

This happens because the specified header's index position does not exist.

## Solution { .section}

Ensure the given index is in the valid range. Index numbering starts from 1.
