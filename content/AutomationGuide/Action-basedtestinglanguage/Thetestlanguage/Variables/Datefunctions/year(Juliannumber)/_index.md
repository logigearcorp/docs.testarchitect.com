# year\(Julian number\) {#year .reference}

Return the Gregorian year from a Julian date number.

## Syntax { .section}

`integer year(integer date)`

## Parameters { .section}

date
:   A Julian date number.

## Return Value { .section}

Returns the corresponding Gregorian year.

## Applicable Built-In Settings { .section}

The following built-in setting\(s\) are supported: [date order](bis_date_order.html).
