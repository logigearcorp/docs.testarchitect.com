# ExpandAll {#abt_ExpandAll .reference}

## Syntax

`long ExpandAll()`

## Description {#section.desc .section}

Display all the nodes of the TreeView control.

## Return Value {#section.return .section}

Return 0 if the call is successful; otherwise, -1..
