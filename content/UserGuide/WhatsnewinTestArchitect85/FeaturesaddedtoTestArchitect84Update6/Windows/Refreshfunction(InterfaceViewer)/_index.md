# Refresh function \(Interface Viewer\) {#Interface_viewer_refresh .concept}

Click the Viewer's **Refresh** button to prompt it to re-inventory all the currently open windows in the system.

If an open interface entity exists, it also reads that and attempts to synchronize it with the corresponding open application window, if any. The **Refresh** ![](../Images/refresh.png) button essentially has the same functionality as closing and re-opening the Interface Viewer.
