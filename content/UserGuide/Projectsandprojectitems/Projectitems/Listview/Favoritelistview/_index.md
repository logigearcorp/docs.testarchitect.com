# Favorite list view {#concept_r5n_jbj_hm .concept}

The favorite list view contains all items \(for example, test modules, actions, data sets, and interfaces\) which you visit frequently and designate as favorites in TestArchitect.

The favorite list view offers the following functionalities:

-   [**Searching**](Listview_filtering.html): Use filters to quickly find an favorite item within the **Favorite** list view.
-   [**Sorting**](Listview_field_sorting.html): Sort your favorites into alphabetical order.
-   [**Opening**](Favorite_items_opening.md#step_rr3_w1h_4l): Conveniently open multiple favorites from the list view.
-   [**Removing**](Favorite_items_removing.html): Remove multiple favorites within the list view.
-   [**Adding all open tabs to favorites**](Favorite_items_adding.md#step_mfb_gbh_4l): Bookmark all open items at once.

-   **[Opening the Favorite list view](../../TA_Help/Topics/Listview_favorites_opening.html)**  
Describes how to open the favorite list view.
