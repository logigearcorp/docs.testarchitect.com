This topic describes the steps required for installing and configuring Oracle Database XE.

1.  On your local machines, download Oracle Database XE at [http://www.oracle.com/technetwork/database/database-technologies/express-edition/downloads/index.html](http://www.oracle.com/technetwork/database/database-technologies/express-edition/downloads/index.html), and then install it.

2.  Follow the below link to see detailed instructions [https://docs.oracle.com/cd/E17781\_01/install.112/e18803/toc.htm\#XEINW123](https://docs.oracle.com/cd/E17781_01/install.112/e18803/toc.htm#XEINW123)
