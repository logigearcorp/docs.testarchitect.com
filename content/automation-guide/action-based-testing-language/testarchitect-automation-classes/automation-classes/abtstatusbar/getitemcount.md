## Syntax

`int GetItemCount()`

## {{< expand >}} Description {{< permerlink >}} {#abt_GetItemCount_status_bar__section.desc} 

Return the number of items in the status bar.

## {{< expand >}} Return Value {{< permerlink >}} {#abt_GetItemCount_status_bar__section.return}
