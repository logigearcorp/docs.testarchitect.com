## {{< expand >}} Description

Enumeration that indicates the mouse button used to click the object.

## Values

**LEFT** or **1**: Clicks the left mouse button

**RIGHT** or **2**: Clicks the right mouse button

**DOUBLE** or **4**: Double-clicks the left mouse button
