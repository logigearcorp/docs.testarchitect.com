## Syntax

`long ClickCenter (TAMouseClickType mouseType)`

## {{< expand >}} Description {{< permerlink >}} {#abt_ClickCenter_element__section.desc} 

Apply a mouse click to the center point of the control or HTML element represented by the object.

## {{< expand >}} Parameters {{< permerlink >}} {#abt_ClickCenter_element__section.params} 

-   **mouseType**

    Type of mouse click.

    Allowable values:

    -   **LEFT or 1**

        Single left-click

    -   **RIGHT or 2**

        Single right-click

    -   **DOUBLE or 4**

        Double left-click


## {{< expand >}} Return Value {{< permerlink >}} {#abt_ClickCenter_element__section.return} 

Return 0 if theclickis successful; otherwise, -1.
