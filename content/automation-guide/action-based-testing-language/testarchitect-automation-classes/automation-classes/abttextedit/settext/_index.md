# SetText

## Syntax

`long SetText(string text)`

## Description

Set the contents of a text control to the specified text data.

## Parameters

-   **text**

    String value specifying the text to be entered into the text box.


## Return Value

Return 0 if the call is successful; otherwise, -1..
