# GetColumnCount

## Syntax

`int GetColumnCount()`

## Description

Retrieve the number of columns in the object.

## Return Value

Integer value indicating the number of columns. A negative value indicatesfailure.

## Notes

-   This method is applicable to ListView controls only.
