# GetSelectedIndex

## Syntax

`int GetSelectedIndex()`

## Description

Retrieve the numeric index of the currently selected item in the object.

## Return Value

Integer index of selected item. \(Note that index numbering begins with 1.\)A negative value indicatesfailure.
