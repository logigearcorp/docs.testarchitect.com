## Syntax

`int GetItemIndex(string text)`

## Description

Retrieve the numeric index of a tab by its text content.

## Parameters

-   **text**

    String value specifying the text content ofthe tab to find.


## Return Value

Integer index of the item if found; otherwise -1.
