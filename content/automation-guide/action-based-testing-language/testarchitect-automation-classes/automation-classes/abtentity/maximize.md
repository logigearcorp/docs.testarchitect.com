## Syntax

`long Maximize()`

## Description

Maximize the window.

## Return Value

Return 0 if the call is successful \(as reported back by the operating system\); otherwise, -1.

## Notes

-   The effect of execution is that the window represented by the AbtEntity object is displayed full-screen.
