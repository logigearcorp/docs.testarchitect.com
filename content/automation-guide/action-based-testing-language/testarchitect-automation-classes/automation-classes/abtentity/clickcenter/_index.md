## Syntax

`long ClickCenter (TAMouseClickType mouseType)`

## Description

Apply a mouse click to the center point of the window.

## Parameters

-   **mouseType**

    Type of mouse click.

    Allowable values:

    -   **LEFT or 1**

        Single left-click

    -   **RIGHT or 2**

        Single right-click

    -   **DOUBLE or 4**

        Double left-click


## Return Value

Return 0 if theclickis successful; otherwise, -1.
