## Syntax

`int GetItemIndex(string itemText)`

## Description

Retrieve the numeric index of a combo box item by its text content.

## Parameters

-   **itemText**

    String value specifying the text content ofitem to find.


## Return Value

Integer index of the item if found; otherwise -1.
