## {{< expand >}} Syntax

`string getCurrentEntityID(string entityType)`

## {{< expand >}} Description

Return the ID of the current project item.

## {{< expand >}} Parameters

-   **entityType**

    Type of the project item.

    Possible values:

    -   testmodule
    -   testcase

## {{< expand >}} Return Value

String value containing ID of the specified project item.
