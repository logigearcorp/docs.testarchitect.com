Convert today’s date to its equivalent Julian day number.

## {{< expand >}} Syntax

`integer today()`

## {{< expand >}} Parameters

`none`

## {{< expand >}} Return Value

Returns today's date as a Julian day number.

## {{< expand >}} Applicable Built-In Settings

The following built-in setting\(s\) are supported: none.
