Return the number of characters in a string.

## {{< expand >}} Syntax

`integer length(string str)`

## {{< expand >}} Parameters

-   **str**

    The source string from which the number of characters is returned.


## {{< expand >}} Return Value

Returns the number of characters in the string str.
